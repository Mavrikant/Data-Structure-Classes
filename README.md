# Linked Listed Class Implementation in C++
- getSize
- push_back
- push_front
- print_list
- search
- deleteLast
- deleteFirst
- deleteIndex
- searchReturnAddr
- deleteValue
- insertAfterIndex

## Compile
* ```git submodule update --init --recursive``` get gtest library
* ```mkdir build```
* ```cd build```
* ```cmake ..```
* ```make all```

## Run Tests
* ```cd /build/tst```
* ```./LinkedList_tst```

## Run main
* ```cd /build/src```
* ```./LinkedList_run```
